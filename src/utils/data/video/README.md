# 视频识别功能说明

## 功能概述
本项目已支持视频识别功能。在视频模式下，系统会：
1. 从本文件夹读取视频文件作为输入
2. 使用虚拟串口模式（IMU数据全部为0）
3. 进行目标检测和识别

## 使用方法

### 1. 准备视频文件
将您的测试视频文件放置在此文件夹中（`src/utils/data/video/`）。
支持的视频格式：`.avi`, `.mp4`, `.mov` 等OpenCV支持的格式。

### 2. 配置文件设置
编辑配置文件 `src/utils/tools/init.json`：

```json
{
  "camera": {
    "device_type": 2,  // 2表示视频模式
    "video_path": "../src/utils/data/video/your_video.avi",  // 修改为您的视频文件名
    ...
  }
}
```

### 3. 运行程序
```bash
cd build
./main
```

## 虚拟串口模式说明
在视频模式下，由于没有真实的串口连接，系统会自动启用虚拟串口模式：
- 所有IMU数据（yaw, pitch, roll）均为0
- 颜色默认为红色（color=101）
- 不会尝试连接真实串口设备
- 不会发送控制指令到串口

## 注意事项
1. 确保视频文件路径正确
2. 视频文件应放在本文件夹中
3. 首次运行时，如果没有视频文件，程序会报错，请先准备好测试视频
4. 建议使用与实际相机分辨率相近的视频进行测试

## 示例视频文件名
- `test.avi` - 默认配置的视频文件名
- `red7.avi` - 红方测试视频
- `blue5.avi` - 蓝方测试视频

## 切换回相机模式
如需切换回使用真实相机，修改配置文件：
```json
{
  "camera": {
    "device_type": 0,  // 0表示大恒相机
    ...
  }
}
